GIRLSTART NONPROFIT WEBSITE REDESIGN

Redesign Girlstart website navigation to increase volunteer sign ups

PROBLEM

Girlstart is a non profit organization needed to provide STEM eduation for young girls. With our initial research and user observations, we found that the website is lacking a clear design path and hierarchial structure that makes the navigation stressful. The volunteer sign up is frustrating and convuluted

HYPOTHESIS

We believe that redesigning Girlstart website to make it more accessible to users will increase in volunteer sign ups for Girlstart.
We will conduct usability tests to see if our revamp of the website has made the process of volunteer sign ups efficient for the users.
We will know the hypoethsis is valid when we get higher success rates in our usability tests than the previous ones.

SOLUTION

Based off of user observations, the solution we thought as a team was to create a more navigable and accessible interface for Girlstart to help increase its user interaction and voulnteer sign up. We can achieve this goal by

Adding a Volunteer Sign Up page for easy navigation.
Prioritize Hierarchyof the content for users to scan the webpage content quickly.
Reassessing color accessibilityof the website that ill include all types of users.

IMPACT

By incorporating acessible color schemes, hierarchyto the webpage content and a clear and more navigable path to volunteer sign up page will increase overall user experienceandenrollment, helping girtlstart to grow and expand.
